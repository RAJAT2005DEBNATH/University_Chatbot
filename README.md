# University Student Support AI Agent

## Description

This project is a Generative AI–powered assistant designed to help university students quickly access important information. The system answers queries about university rules, departments, courses, and events. It also analyzes student feedback and generates summaries or simple reports using a Retrieval-Augmented Generation (RAG) pipeline with the Gemini API.

## Features

- Answer questions related to university rules, courses, exams, and events
- Retrieve relevant information from stored university documents using RAG
- Summarize student feedback from survey data
- Generate simple reports highlighting student issues and feedback trends
- Interactive web interface built with Streamlit

## Tech Stack

- **LLM:** Google Gemini API
- **Framework:** LangChain
- **Embeddings:** sentence-transformers
- **Vector Database:** ChromaDB / FAISS
- **Frontend:** Streamlit
- **Language:** Python
